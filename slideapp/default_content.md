<!-- 常用语法： -->

<!-- 0.支持markdown基础语法 -->

<!--
1.使用 --- （三个减号的单独行）来划分水平幻灯片
-->

<!--
2.使用 ----（四个减号的单独行）来划分垂直幻灯片
-->

<!--
3.使用关键字 ++++ 来划分渐变幻灯片
-->

<!--
4.可以直接粘贴或者拖动图片到输入框，网页会自动上传图片到服务器，并且自动补全markdown格式的图片
-->

<!--
5.

图片居中 <img alt=... class="center" src=...>

自定义图片大小<img alt=... src=... width="678px">
-->

<!--
6.代码高亮，在三个反引号的后面增加语言类型：

例如：

```c++
#include <iostream>

int main() {
    std::cout << "Hello World!" << std::endl;

    return 0;
}
```
-->

<!--
7.左右分栏（columns）：单张幻灯片内使用 ::: columns 比例 ... :::

例如：

::: columns 40/60
::: column
左侧 Markdown
- 列表
- 文字
:::
::: column
右侧 Markdown 或图片

![](图片路径)
:::
:::

比例支持任意列数与任意正整数比，如 1/2、30/30/40。
省略比例则均分。比例必须写在同一张幻灯片内，不能跨 --- / ----。
-->

<!--
8.其他块级指令（同一张幻灯片内闭合）：

::: tip 关键字后可写自定义标题
- 五种提示框：note 📝、tip 💡、warning ⚠️、danger 🚫、success ✅
:::

::: notes
演讲者备注（听众看不到；目前是"隐藏但保留"，未来挂 RevealNotes 插件可在第二屏看）
:::

::: incremental
- 第一条
- 第二条
- 第三条（每条变成 fragment，按空格逐条出现）
:::

::: timeline
- 2024.01: MVP 上线
- 2024.06: 大改版
:::
-->

论断[^1]脚注（文档末尾对应 [^1]: 出处）
